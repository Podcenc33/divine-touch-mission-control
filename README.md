# Divine Touch Mission Control

**Unified AI Office Manager Command Center** for Divine Touch Barbershop.

This repository serves as the central hub for the autonomous AI team (Hermes, Client Memory, Morning Report, Budget Tracker, Content & Video, etc.).

### Core Components
- **Visual Mission Control** – Interactive 2D barbershop dashboard (click agents for live status)
- **CLI Launcher** – `mc` command that prints briefing + opens visual
- **Integration Hub** – Connects all agents (Client profiles, budget tracking, daily briefings)
- **Persistent Memory** – Real client data (Brian Thompson, Priya Patel, Evelyn Park, Alex Rivera, etc.)

The system runs autonomously with daily 6:30 AM voice + text briefings.

**Current Visual Dashboard:** `~/.hermes/2d_office/index.html`

**Launcher:** Run `mc` from anywhere.
